# Mandelbrot Explorer

A simple [Mandelbrot](https://en.wikipedia.org/wiki/Mandelbrot_set) fractal explorer written in processing.

### Controls

|                        |             |
| ---------------------- | ----------- |
| Panning                | w, a, s, d  |
| Zooming                | up, down    |
| Change Iteration Count | right, left |
| Save Image             | p           |
